The morning command center - 20 variants

One product, twenty directions.

Twenty takes on the same morning screen, each a fusion: one visual language (from the inspiration set and the three skins) crossed with one interaction or feature and a motion or AI-state signature. All light, all calm, all checked against the anti-slop rules. Built to compare, so the strongest direction can be chosen.
