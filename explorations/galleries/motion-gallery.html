Motion catalog / Leo command center

Every animation orients, gives feedback, or encodes an AI confidence state.

Motion here is never decoration. It is the channel that tells you what Leo is doing, how sure it is, and whether it needs you. All of it is CSS keyframes and transitions, not JavaScript, so it degrades gracefully: if the script never runs, every demo still shows its resting state. Tap replay on any tile to watch it again. Easing is a single ease-out curve throughout; reduced-motion users get the final frame, instantly.
