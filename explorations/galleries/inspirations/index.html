The morning command center - visual range

One morning, ten visual languages.

Each of these is the same product, the Leo morning command center, rebuilt from scratch in a different current app aesthetic. Same content, same job: surface the three deals that need Dana this morning. The point is range: a Lead designer should be able to dress one idea in ten visual languages, then choose a restrained one on purpose, not because it is all they can do.

Each screen is a self-contained, hand-coded prototype. Motion is CSS, not JavaScript, so everything renders and animates even under a strict content-security policy. None of these is the product. The product is the calm one.
